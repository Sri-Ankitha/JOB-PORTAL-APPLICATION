JOB PORTAL APPLICATION

A complete Job Portal Web Application designed for both Students (Job Seekers) and Employers (Recruiters).
Employers can post jobs and track applicants, while students can browse and apply for jobs easily.

Features
Employer Features

Post new job openings

View applications submitted by students

Edit or delete job posts

Manage employer dashboard

View student resumes

Student Features

Student registration & login

Browse job listings

Apply for jobs

Upload resume

Student dashboard access

View application status

Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: Python Flask

Database: SQLite / MySQL

Version Control: Git & GitHub

Installation & Setup
1. Clone the Repository
git clone https://github.com/Sri-Ankitha/JOB-PORTAL-APPLICATION.git
cd JOB-PORTAL-APPLICATION

2. Install Dependencies
pip install -r requirements.txt

3. Run the Application
python app.py


Application runs at:

http://127.0.0.1:5000/
